# v0.1.7

## Highlights
- Omarchy theme integration for seamless desktop-wide color consistency.
- CI pipeline enforces tests and linting on every pull request.
- Auto-release now triggers correctly on merges to `master`.

## Added
- Omarchy theme adapter that reads the active desktop theme and maps it to omakure's palette.
- CI workflow with `cargo test` and `cargo clippy`/`cargo fmt --check` as required PR checks.
- Project architecture and requirements knowledge base in `.docs/`.
- Workflow templates for requirements, plans, and summaries.

## Changed
- Auto-release workflow targets `master` instead of `main`.
- Theme CLI commands updated to support omarchy theme source.

## Chores
- Added `.claude` to `.gitignore` and configured mise tasks.
- Refined auto-release trigger to fire on merged pull requests.
